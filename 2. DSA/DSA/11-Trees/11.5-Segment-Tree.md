# 11.5 Segment Tree

A Segment Tree answers range queries (sum, min, max) in O(log n) after O(n) build time, and supports point updates in O(log n). It is the go-to structure for competitive programming range problems.

## Introduction

### What is a Segment Tree?

A **Segment Tree** is a binary tree where:
- Each leaf stores a single array element.
- Each internal node stores the aggregate (sum / min / max) of a range.

```
Array: [1, 3, 5, 7, 9, 11]

              [0,5]=36
             /          \
       [0,2]=9         [3,5]=27
       /     \          /     \
   [0,1]=4  [2,2]=5 [3,4]=16 [5,5]=11
   /    \            /    \
[0,0]=1 [1,1]=3  [3,3]=7 [4,4]=9
```

### When to Use

| Problem Type | Structure |
|-------------|-----------|
| Range sum query + point update | Segment Tree |
| Range min/max + point update | Segment Tree |
| Range updates + range queries | Segment Tree with Lazy Propagation |
| Static range queries only | Sparse Table (O(1) query) |

## Structure and Representation

### Array-Based (1-indexed)

```
For node at index i:
  Left child  = 2 * i
  Right child = 2 * i + 1
  Parent      = i / 2

Array size needed: 4 * n  (safe upper bound)

tree[1]           = root (covers entire array)
tree[2], tree[3]  = children of root
...
```

```java
int n;
int[] tree;

public SegmentTree(int[] arr) {
    n = arr.length;
    tree = new int[4 * n];
    build(arr, 1, 0, n - 1);
}
```

## Build

```java
// Build segment tree recursively
// node: current tree index
// start, end: range this node covers in original array
private void build(int[] arr, int node, int start, int end) {
    if (start == end) {
        tree[node] = arr[start];  // leaf
        return;
    }

    int mid = (start + end) / 2;
    build(arr, 2 * node,     start, mid);      // build left
    build(arr, 2 * node + 1, mid + 1, end);    // build right

    tree[node] = tree[2 * node] + tree[2 * node + 1];  // merge
}
// Time: O(n) | Space: O(n) for tree array
```

### Build Trace

```
Array: [1, 3, 5, 7]  n=4

build(1, 0, 3):
  build(2, 0, 1):
    build(4, 0, 0): tree[4] = 1
    build(5, 1, 1): tree[5] = 3
    tree[2] = 1 + 3 = 4
  build(3, 2, 3):
    build(6, 2, 2): tree[6] = 5
    build(7, 3, 3): tree[7] = 7
    tree[3] = 5 + 7 = 12
  tree[1] = 4 + 12 = 16

tree = [_, 16, 4, 12, 1, 3, 5, 7]
         ↑ index 0 unused (1-indexed)
```

## Range Query

```java
// Query sum in range [l, r]
// node: current node, [start, end]: node's range, [l, r]: query range
public int query(int node, int start, int end, int l, int r) {
    if (r < start || end < l) {
        return 0;  // completely outside: identity for sum (0), MAX_VALUE for min
    }
    if (l <= start && end <= r) {
        return tree[node];  // completely inside: return stored value
    }
    int mid = (start + end) / 2;
    int left  = query(2 * node,     start, mid,     l, r);
    int right = query(2 * node + 1, mid + 1, end,   l, r);
    return left + right;
}

// Wrapper
public int query(int l, int r) {
    return query(1, 0, n - 1, l, r);
}
```

### Query Cases

```
Query [1, 3] on tree covering [0, 3]:

         [0,3]
         /   \
      [0,1]  [2,3]
      /  \   /  \
   [0,0][1,1][2,2][3,3]

Query reaches [0,1]: l=1 ≤ 0? No → split
  [0,0]: 0 < l=1 → outside, return 0
  [1,1]: fully inside [1,3] → return tree value
[2,3]: fully inside [1,3] → return tree value

Sum = 0 + tree[1,1] + tree[2,3]
```

## Point Update

```java
// Update arr[idx] to newVal
public void update(int node, int start, int end, int idx, int newVal) {
    if (start == end) {
        tree[node] = newVal;  // update leaf
        return;
    }
    int mid = (start + end) / 2;
    if (idx <= mid) update(2 * node,     start, mid,     idx, newVal);
    else            update(2 * node + 1, mid + 1, end,   idx, newVal);

    tree[node] = tree[2 * node] + tree[2 * node + 1];  // re-merge
}

// Wrapper
public void update(int idx, int newVal) {
    update(1, 0, n - 1, idx, newVal);
}
// Time: O(log n) — only one root-to-leaf path is updated
```

## Range Sum Query Template

Full reusable class:

```java
public class SegmentTree {
    private final int[] tree;
    private final int n;

    public SegmentTree(int[] arr) {
        n = arr.length;
        tree = new int[4 * n];
        build(arr, 1, 0, n - 1);
    }

    private void build(int[] arr, int node, int s, int e) {
        if (s == e) { tree[node] = arr[s]; return; }
        int m = (s + e) / 2;
        build(arr, 2*node,   s, m);
        build(arr, 2*node+1, m+1, e);
        tree[node] = tree[2*node] + tree[2*node+1];
    }

    public void update(int idx, int val) {
        update(1, 0, n-1, idx, val);
    }

    private void update(int node, int s, int e, int idx, int val) {
        if (s == e) { tree[node] = val; return; }
        int m = (s + e) / 2;
        if (idx <= m) update(2*node,   s, m,   idx, val);
        else          update(2*node+1, m+1, e, idx, val);
        tree[node] = tree[2*node] + tree[2*node+1];
    }

    public int query(int l, int r) {
        return query(1, 0, n-1, l, r);
    }

    private int query(int node, int s, int e, int l, int r) {
        if (r < s || e < l) return 0;
        if (l <= s && e <= r) return tree[node];
        int m = (s + e) / 2;
        return query(2*node, s, m, l, r)
             + query(2*node+1, m+1, e, l, r);
    }
}
```

## Range Min / Max Query

```java
public class SegmentTreeMin {
    private final int[] tree;
    private final int n;

    public SegmentTreeMin(int[] arr) {
        n = arr.length;
        tree = new int[4 * n];
        build(arr, 1, 0, n - 1);
    }

    private void build(int[] arr, int node, int s, int e) {
        if (s == e) { tree[node] = arr[s]; return; }
        int m = (s + e) / 2;
        build(arr, 2*node,   s, m);
        build(arr, 2*node+1, m+1, e);
        tree[node] = Math.min(tree[2*node], tree[2*node+1]);
    }

    public void update(int idx, int val) { update(1, 0, n-1, idx, val); }

    private void update(int nd, int s, int e, int idx, int val) {
        if (s == e) { tree[nd] = val; return; }
        int m = (s + e) / 2;
        if (idx <= m) update(2*nd,   s, m,   idx, val);
        else          update(2*nd+1, m+1, e, idx, val);
        tree[nd] = Math.min(tree[2*nd], tree[2*nd+1]);
    }

    public int queryMin(int l, int r) { return queryMin(1, 0, n-1, l, r); }

    private int queryMin(int nd, int s, int e, int l, int r) {
        if (r < s || e < l) return Integer.MAX_VALUE;  // identity for min
        if (l <= s && e <= r) return tree[nd];
        int m = (s + e) / 2;
        return Math.min(queryMin(2*nd, s, m, l, r),
                        queryMin(2*nd+1, m+1, e, l, r));
    }
}
```

### Identity Values by Operation

| Operation | Identity (out-of-range return) |
|-----------|-------------------------------|
| Sum | 0 |
| Min | Integer.MAX_VALUE |
| Max | Integer.MIN_VALUE |
| GCD | 0 |
| Product | 1 |

## Common Problems

### Problem 1: Range Sum Query — Mutable ⭐⭐⭐⭐⭐

**LeetCode 307** | Medium

```java
class NumArray {
    private final SegmentTree st;

    public NumArray(int[] nums) {
        st = new SegmentTree(nums);
    }

    public void update(int index, int val) {
        st.update(index, val);
    }

    public int sumRange(int left, int right) {
        return st.query(left, right);
    }
}
// Time: O(log n) per update and query
```

### Problem 2: Count of Range Sum ⭐⭐⭐⭐

**LeetCode 327** | Hard

**Approach:** Build segment tree on compressed prefix sums.

```java
// Coordinate compression + segment tree or merge sort approach
public int countRangeSum(int[] nums, int lower, int upper) {
    int n = nums.length;
    long[] prefix = new long[n + 1];
    for (int i = 0; i < n; i++) prefix[i+1] = prefix[i] + nums[i];
    return mergeCount(prefix, 0, n, lower, upper);
}

private int mergeCount(long[] arr, int lo, int hi, int lower, int upper) {
    if (hi - lo <= 0) return 0;
    int mid = (lo + hi) / 2;
    int count = mergeCount(arr, lo, mid, lower, upper)
              + mergeCount(arr, mid+1, hi, lower, upper);

    int j = mid+1, k = mid+1;
    for (int i = lo; i <= mid; i++) {
        while (j <= hi && arr[j] - arr[i] < lower) j++;
        while (k <= hi && arr[k] - arr[i] <= upper) k++;
        count += k - j;
    }

    // Merge step
    long[] temp = Arrays.copyOfRange(arr, lo, hi+1);
    int p = 0, q = mid-lo+1, r = lo;
    while (p <= mid-lo && q < temp.length) {
        arr[r++] = temp[p] <= temp[q] ? temp[p++] : temp[q++];
    }
    while (p <= mid-lo) arr[r++] = temp[p++];
    while (q < temp.length) arr[r++] = temp[q++];

    return count;
}
// Time: O(n log n) | Space: O(n)
```

### Problem 3: Falling Squares ⭐⭐⭐

**LeetCode 699** | Hard

```java
// Segment tree with coordinate compression for range max queries
public List<Integer> fallingSquares(int[][] positions) {
    List<Integer> result = new ArrayList<>();
    // Simulate with interval tracking (simpler O(n²))
    int[] heights = new int[positions.length];
    int maxHeight = 0;

    for (int i = 0; i < positions.length; i++) {
        int left = positions[i][0], size = positions[i][1];
        int right = left + size;
        heights[i] = size;

        for (int j = 0; j < i; j++) {
            int lj = positions[j][0], rj = positions[j][0] + positions[j][1];
            if (left < rj && right > lj) {
                heights[i] = Math.max(heights[i], heights[j] + size);
            }
        }
        maxHeight = Math.max(maxHeight, heights[i]);
        result.add(maxHeight);
    }
    return result;
}
```

## Key Takeaways

- Segment tree array size = `4 * n` (1-indexed, 2*n for 0-indexed iterative).
- Three operations: `build O(n)`, `query O(log n)`, `update O(log n)`.
- Query logic: if range is completely outside → return identity; if completely inside → return stored value; else recurse on both halves.
- Swap the merge function to switch from sum to min/max/gcd/product.
- For range updates (not just point), add lazy propagation (see 19.1-Segment-Tree-Advanced.md).
- In competitive programming, use 1-indexed tree array and `tree[2*node]` / `tree[2*node+1]` for children.
