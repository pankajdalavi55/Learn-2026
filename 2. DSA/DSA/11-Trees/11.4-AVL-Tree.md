# 11.4 AVL Tree

An AVL tree guarantees O(log n) worst-case for all operations by maintaining the balance factor invariant. Understanding rotations is the key to implementing self-balancing trees and to understanding the theory behind Java's `TreeMap`.

## What is an AVL Tree?

An **AVL Tree** (Adelson-Velsky and Landis, 1962) is a self-balancing BST where the **height difference** between left and right subtrees of every node is at most 1.

```
Balanced AVL:           Unbalanced (not AVL):
       10                      10
      /  \                    /
     5    15                 5
    / \     \               /
   3   7    20             3
                          /
                         1
BF(10) = h(L) - h(R)    BF(10) = 3 - 0 = 3  ← violation!
       = 2 - 1 = 1  ✓
```

### Why AVL?

| Tree Type | Search | Insert | Delete |
|-----------|--------|--------|--------|
| Unbalanced BST | O(n) worst | O(n) worst | O(n) worst |
| AVL Tree | O(log n) | O(log n) | O(log n) |
| Red-Black Tree | O(log n) | O(log n) | O(log n) |

## Balance Factor

```
Balance Factor (BF) = height(left subtree) - height(right subtree)

Valid BF values: {-1, 0, 1}
If |BF| > 1 → rotate to restore balance

BF = +2 → left heavy → rotate right
BF = -2 → right heavy → rotate left
```

```java
private int height(AVLNode node) {
    return node == null ? 0 : node.height;
}

private int balanceFactor(AVLNode node) {
    return node == null ? 0 : height(node.left) - height(node.right);
}

private void updateHeight(AVLNode node) {
    node.height = 1 + Math.max(height(node.left), height(node.right));
}
```

## Rotations

### Right Rotation (LL Case)

**When:** BF(node) = +2 and BF(node.left) >= 0

```
Before:              After:
    z                   y
   /                   / \
  y        →          x   z
 /
x

Right rotate z:
  y.right = z
  z.left  = y.right (old)
  return y
```

```java
private AVLNode rotateRight(AVLNode z) {
    AVLNode y = z.left;
    AVLNode T3 = y.right;   // subtree between y and z

    y.right = z;
    z.left  = T3;

    updateHeight(z);
    updateHeight(y);

    return y;   // y is new root
}
```

### Left Rotation (RR Case)

**When:** BF(node) = -2 and BF(node.right) <= 0

```
Before:              After:
  z                     y
   \                   / \
    y       →          z   x
     \
      x

Left rotate z:
  y.left  = z
  z.right = y.left (old)
  return y
```

```java
private AVLNode rotateLeft(AVLNode z) {
    AVLNode y = z.right;
    AVLNode T2 = y.left;

    y.left  = z;
    z.right = T2;

    updateHeight(z);
    updateHeight(y);

    return y;
}
```

### Left-Right Rotation (LR Case)

**When:** BF(node) = +2 and BF(node.left) < 0

```
Before:              After Left on y:    After Right on z:
    z                    z                   x
   /                    /                   / \
  y          →          x          →       y   z
   \                   /
    x                 y
```

```java
private AVLNode rotateLeftRight(AVLNode z) {
    z.left = rotateLeft(z.left);   // left rotate y
    return rotateRight(z);          // right rotate z
}
```

### Right-Left Rotation (RL Case)

**When:** BF(node) = -2 and BF(node.right) > 0

```
Before:              After Right on y:   After Left on z:
  z                      z                    x
   \                      \                  / \
    y          →            x       →       z   y
   /                         \
  x                           y
```

```java
private AVLNode rotateRightLeft(AVLNode z) {
    z.right = rotateRight(z.right);  // right rotate y
    return rotateLeft(z);             // left rotate z
}
```

### Rotation Decision Table

| BF(z) | BF(child) | Case | Rotation |
|-------|-----------|------|----------|
| +2 | >= 0 | LL | Right rotate z |
| +2 | < 0 | LR | Left rotate z.left, then right rotate z |
| -2 | <= 0 | RR | Left rotate z |
| -2 | > 0 | RL | Right rotate z.right, then left rotate z |

## Insertion

```java
private AVLNode insert(AVLNode node, int key) {
    // 1. Standard BST insert
    if (node == null) return new AVLNode(key);

    if (key < node.key)       node.left  = insert(node.left,  key);
    else if (key > node.key)  node.right = insert(node.right, key);
    else                      return node;  // duplicate

    // 2. Update height
    updateHeight(node);

    // 3. Get balance factor
    int bf = balanceFactor(node);

    // 4. Restore balance with rotations
    // LL Case
    if (bf > 1 && key < node.left.key)
        return rotateRight(node);

    // RR Case
    if (bf < -1 && key > node.right.key)
        return rotateLeft(node);

    // LR Case
    if (bf > 1 && key > node.left.key) {
        node.left = rotateLeft(node.left);
        return rotateRight(node);
    }

    // RL Case
    if (bf < -1 && key < node.right.key) {
        node.right = rotateRight(node.right);
        return rotateLeft(node);
    }

    return node;
}
```

### Insertion Trace

```
Insert 30, 20, 10 into empty AVL:

After insert(30):   30 (BF=0)

After insert(20):   30 (BF=1)
                   /
                  20

After insert(10):   30 (BF=2) ← LL Case! → right rotate 30
                   /
                  20 (BF=1)
                 /
                10

After rotation:   20
                 /  \
                10   30   (balanced!)
```

## Deletion

```java
private AVLNode delete(AVLNode node, int key) {
    // 1. Standard BST delete
    if (node == null) return null;

    if (key < node.key) {
        node.left = delete(node.left, key);
    } else if (key > node.key) {
        node.right = delete(node.right, key);
    } else {
        if (node.left == null)  return node.right;
        if (node.right == null) return node.left;

        // Replace with inorder successor
        AVLNode successor = findMin(node.right);
        node.key = successor.key;
        node.right = delete(node.right, successor.key);
    }

    // 2. Update height
    updateHeight(node);

    // 3. Rebalance
    int bf = balanceFactor(node);

    if (bf > 1 && balanceFactor(node.left) >= 0)
        return rotateRight(node);

    if (bf > 1 && balanceFactor(node.left) < 0) {
        node.left = rotateLeft(node.left);
        return rotateRight(node);
    }

    if (bf < -1 && balanceFactor(node.right) <= 0)
        return rotateLeft(node);

    if (bf < -1 && balanceFactor(node.right) > 0) {
        node.right = rotateRight(node.right);
        return rotateLeft(node);
    }

    return node;
}

private AVLNode findMin(AVLNode node) {
    while (node.left != null) node = node.left;
    return node;
}
```

## Complete AVL Tree Implementation

```java
public class AVLTree {

    private static class AVLNode {
        int key, height;
        AVLNode left, right;

        AVLNode(int key) {
            this.key = key;
            this.height = 1;
        }
    }

    private AVLNode root;

    private int height(AVLNode n) { return n == null ? 0 : n.height; }

    private void updateHeight(AVLNode n) {
        n.height = 1 + Math.max(height(n.left), height(n.right));
    }

    private int bf(AVLNode n) { return n == null ? 0 : height(n.left) - height(n.right); }

    private AVLNode rotateRight(AVLNode z) {
        AVLNode y = z.left;
        z.left = y.right;
        y.right = z;
        updateHeight(z);
        updateHeight(y);
        return y;
    }

    private AVLNode rotateLeft(AVLNode z) {
        AVLNode y = z.right;
        z.right = y.left;
        y.left = z;
        updateHeight(z);
        updateHeight(y);
        return y;
    }

    private AVLNode balance(AVLNode n) {
        updateHeight(n);
        int b = bf(n);

        if (b > 1) {
            if (bf(n.left) < 0) n.left = rotateLeft(n.left);
            return rotateRight(n);
        }
        if (b < -1) {
            if (bf(n.right) > 0) n.right = rotateRight(n.right);
            return rotateLeft(n);
        }
        return n;
    }

    private AVLNode insert(AVLNode n, int key) {
        if (n == null) return new AVLNode(key);
        if (key < n.key)      n.left  = insert(n.left,  key);
        else if (key > n.key) n.right = insert(n.right, key);
        return balance(n);
    }

    private AVLNode delete(AVLNode n, int key) {
        if (n == null) return null;
        if (key < n.key)      n.left  = delete(n.left,  key);
        else if (key > n.key) n.right = delete(n.right, key);
        else {
            if (n.left == null)  return n.right;
            if (n.right == null) return n.left;
            AVLNode min = n.right;
            while (min.left != null) min = min.left;
            n.key = min.key;
            n.right = delete(n.right, min.key);
        }
        return balance(n);
    }

    public void insert(int key) { root = insert(root, key); }
    public void delete(int key) { root = delete(root, key); }
    public boolean contains(int key) {
        AVLNode n = root;
        while (n != null) {
            if (key == n.key) return true;
            n = key < n.key ? n.left : n.right;
        }
        return false;
    }
    public int height() { return height(root); }
}
```

### Complexity Summary

| Operation | Time | Space |
|-----------|------|-------|
| Insert | O(log n) | O(log n) — recursion |
| Delete | O(log n) | O(log n) |
| Search | O(log n) | O(1) iterative |
| Height | O(1) — stored | O(1) |

## AVL vs Red-Black Tree

| Property | AVL | Red-Black |
|----------|-----|-----------|
| Balance Guarantee | Strictly balanced (|BF| ≤ 1) | Roughly balanced (h ≤ 2 log n) |
| Height | log₂ n | ≤ 2 log₂(n+1) |
| Search Speed | Faster (lower height) | Slightly slower |
| Insert/Delete | More rotations | Fewer rotations |
| Use Case | Read-heavy workloads | Write-heavy workloads |
| Java Usage | Not directly in JDK | TreeMap, TreeSet (Java 8+) |

---

## Key Takeaways

- AVL property: `|height(left) - height(right)| ≤ 1` at every node.
- Four rotation cases: LL (right), RR (left), LR (left then right), RL (right then left).
- Store height in each node and update it bottom-up after insert/delete.
- A single insertion causes at most 1 rotation (at the first unbalanced ancestor).
- A single deletion may cause rotations all the way up the path — O(log n) rotations possible.
- Use `TreeMap`/`TreeSet` in Java for O(log n) ordered operations instead of implementing AVL.
