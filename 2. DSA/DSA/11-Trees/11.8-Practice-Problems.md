## Pattern Cheat Sheet

### Pattern 1: Post-Order Aggregation (most common)

```java
// Template: compute something for subtree, combine, return up
int solve(TreeNode node) {
    if (node == null) return BASE_CASE;
    int left  = solve(node.left);
    int right = solve(node.right);
    // update global answer if needed
    globalAnswer = max(globalAnswer, f(left, right, node.val));
    return g(left, right, node.val);  // return to parent
}

// Problems: diameter, max path sum, height, balanced check
```

### Pattern 2: BFS Level-by-Level

```java
// Template
Queue<TreeNode> queue = new LinkedList<>();
queue.offer(root);

while (!queue.isEmpty()) {
    int size = queue.size();  // freeze current level size
    for (int i = 0; i < size; i++) {
        TreeNode node = queue.poll();
        // process node
        if (node.left  != null) queue.offer(node.left);
        if (node.right != null) queue.offer(node.right);
    }
}

// Problems: level order, right side view, zigzag, min depth
```

### Pattern 3: BST Property Exploit

```java
// Template: use min/max bounds to validate or search
boolean validate(TreeNode node, long min, long max) {
    if (node == null) return true;
    if (node.val <= min || node.val >= max) return false;
    return validate(node.left,  min, node.val)
        && validate(node.right, node.val, max);
}

// If target < node.val → go left only
// If target > node.val → go right only

// Problems: validate BST, LCA in BST, floor/ceiling, search
```

### Pattern 4: DFS with Backtracking (Path Problems)

```java
// Template
void dfs(TreeNode node, int remaining, List<Integer> path, List<List<Integer>> result) {
    if (node == null) return;
    path.add(node.val);
    if (node.left == null && node.right == null && remaining == node.val) {
        result.add(new ArrayList<>(path));  // found valid path
    }
    dfs(node.left,  remaining - node.val, path, result);
    dfs(node.right, remaining - node.val, path, result);
    path.remove(path.size() - 1);  // backtrack
}

// Problems: Path Sum II, root-to-leaf paths
```

### Pattern 5: Tree DP (Bottom-Up State)

```java
// Return pair/array of states to parent
int[] dp(TreeNode node) {
    if (node == null) return new int[]{0, 0};  // {rob, skip}
    int[] left  = dp(node.left);
    int[] right = dp(node.right);
    int rob  = node.val + left[1] + right[1];  // rob this node
    int skip = Math.max(left[0], left[1]) + Math.max(right[0], right[1]);
    return new int[]{rob, skip};
}

// Problems: House Robber III, Binary Tree Cameras
```

## Solutions Walkthrough

### Binary Tree Maximum Path Sum (LC 124) ⭐⭐⭐⭐⭐

```java
private int maxSum = Integer.MIN_VALUE;

public int maxPathSum(TreeNode root) {
    maxSum = Integer.MIN_VALUE;
    gainFromNode(root);
    return maxSum;
}

private int gainFromNode(TreeNode node) {
    if (node == null) return 0;

    int leftGain  = Math.max(gainFromNode(node.left),  0);  // ignore negative
    int rightGain = Math.max(gainFromNode(node.right), 0);

    // Path through this node
    int pathThroughNode = node.val + leftGain + rightGain;
    maxSum = Math.max(maxSum, pathThroughNode);

    // Return max single-branch gain to parent
    return node.val + Math.max(leftGain, rightGain);
}
// Time: O(n) | Space: O(h)
```

**Key insight:** The path can go left→node→right (not passed up), but only one branch can be returned to parent.

### Serialize and Deserialize Binary Tree (LC 297) ⭐⭐⭐⭐⭐

```java
private static final String NULL = "null", SEP = ",";

public String serialize(TreeNode root) {
    StringBuilder sb = new StringBuilder();
    serializeHelper(root, sb);
    return sb.toString();
}

private void serializeHelper(TreeNode node, StringBuilder sb) {
    if (node == null) { sb.append(NULL).append(SEP); return; }
    sb.append(node.val).append(SEP);
    serializeHelper(node.left, sb);
    serializeHelper(node.right, sb);
}

public TreeNode deserialize(String data) {
    Queue<String> queue = new LinkedList<>(Arrays.asList(data.split(SEP)));
    return deserializeHelper(queue);
}

private TreeNode deserializeHelper(Queue<String> queue) {
    String val = queue.poll();
    if (NULL.equals(val)) return null;
    TreeNode node = new TreeNode(Integer.parseInt(val));
    node.left  = deserializeHelper(queue);
    node.right = deserializeHelper(queue);
    return node;
}
// Time: O(n) | Space: O(n)
```

### Construct Binary Tree from Preorder and Inorder (LC 105) ⭐⭐⭐⭐⭐

```java
public TreeNode buildTree(int[] preorder, int[] inorder) {
    Map<Integer, Integer> inMap = new HashMap<>();
    for (int i = 0; i < inorder.length; i++) inMap.put(inorder[i], i);
    return build(preorder, 0, preorder.length - 1, 0, inorder.length - 1, inMap);
}

private TreeNode build(int[] pre, int preL, int preR,
                       int inL, int inR, Map<Integer, Integer> inMap) {
    if (preL > preR) return null;
    int rootVal = pre[preL];
    TreeNode root = new TreeNode(rootVal);
    int inIdx = inMap.get(rootVal);
    int leftSize = inIdx - inL;

    root.left  = build(pre, preL + 1, preL + leftSize, inL, inIdx - 1, inMap);
    root.right = build(pre, preL + leftSize + 1, preR, inIdx + 1, inR, inMap);
    return root;
}
// Time: O(n) | Space: O(n)
```

### House Robber III (LC 337) ⭐⭐⭐⭐

```java
public int rob(TreeNode root) {
    int[] res = robHelper(root);
    return Math.max(res[0], res[1]);
}

// Returns [rob_this_node, skip_this_node]
private int[] robHelper(TreeNode node) {
    if (node == null) return new int[]{0, 0};

    int[] left  = robHelper(node.left);
    int[] right = robHelper(node.right);

    int rob  = node.val + left[1] + right[1];
    int skip = Math.max(left[0], left[1]) + Math.max(right[0], right[1]);

    return new int[]{rob, skip};
}
// Time: O(n) | Space: O(h)
```

### All Nodes Distance K (LC 863) ⭐⭐⭐⭐

```java
public List<Integer> distanceK(TreeNode root, TreeNode target, int k) {
    // Build parent map, then BFS from target
    Map<TreeNode, TreeNode> parentMap = new HashMap<>();
    buildParentMap(root, null, parentMap);

    List<Integer> result = new ArrayList<>();
    Set<TreeNode> visited = new HashSet<>();
    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(target);
    visited.add(target);

    int dist = 0;
    while (!queue.isEmpty()) {
        if (dist == k) {
            for (TreeNode n : queue) result.add(n.val);
            return result;
        }
        int size = queue.size();
        for (int i = 0; i < size; i++) {
            TreeNode node = queue.poll();
            for (TreeNode neighbor : new TreeNode[]{node.left, node.right, parentMap.get(node)}) {
                if (neighbor != null && !visited.contains(neighbor)) {
                    visited.add(neighbor);
                    queue.offer(neighbor);
                }
            }
        }
        dist++;
    }
    return result;
}

private void buildParentMap(TreeNode node, TreeNode parent,
                            Map<TreeNode, TreeNode> map) {
    if (node == null) return;
    map.put(node, parent);
    buildParentMap(node.left, node, map);
    buildParentMap(node.right, node, map);
}
// Time: O(n) | Space: O(n)
```

## Key Takeaways

1. **Post-order** is the most common tree pattern: compute for children, then decide for current node.
2. **BFS with `levelSize`** is the template for all level-based problems.
3. **BST problems**: always exploit `left < node < right` — avoid visiting unnecessary nodes.
4. **Path sum problems**: DFS with remaining target; backtrack by removing from path list.
5. **Tree DP**: return a tuple/array of states when you need both "include" and "exclude" decisions.
6. **LCA**: post-order — if both sides return non-null, current is the answer.
7. **Serialize/deserialize**: preorder + null markers gives a unique encoding.
8. **Distance K**: build parent map first to enable upward BFS traversal.
