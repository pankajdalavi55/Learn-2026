# 11.7 Trie

A Trie (prefix tree) stores strings character by character, making prefix searches O(L) regardless of dictionary size. It is the backbone of autocomplete, spell-check, and IP routing.

## Introduction

### What is a Trie?

A **Trie** is a tree where each node represents a character, and paths from root to marked nodes spell out stored words.

```
Words: ["apple", "app", "apt", "bat", "ball"]

             root
            /    \
           a      b
           |      |
           p      a
          / \    / \
         p   t  t   l
         |   |  |   |
         l   *  *   l
         |          |
         e          *
         |
         *     (* = end of word)
```

### Why Trie?

| Operation | HashMap | Trie |
|-----------|---------|------|
| Insert word | O(L) | O(L) |
| Search word | O(L) | O(L) |
| Prefix search | O(n*L) — linear scan | O(L) |
| All words with prefix | O(n*L) | O(L + output) |
| Memory | O(n*L) | O(n*L) worst, shared prefixes save space |

Where L = word length, n = number of words.

## Trie Node Structure

### Array-Based (26 lowercase letters)

```java
class TrieNode {
    TrieNode[] children;
    boolean isEnd;

    TrieNode() {
        children = new TrieNode[26];  // 'a' to 'z'
        isEnd = false;
    }
}
```

### HashMap-Based (any character set)

```java
class TrieNode {
    Map<Character, TrieNode> children;
    boolean isEnd;

    TrieNode() {
        children = new HashMap<>();
        isEnd = false;
    }
}
```

**Trade-off:**

| | Array | HashMap |
|-|-------|---------|
| Access | O(1) | O(1) avg |
| Memory | 26 pointers per node (even if unused) | Only used keys |
| Use case | Lowercase English only | Unicode/variable alphabets |

## Insert

```java
public void insert(String word) {
    TrieNode curr = root;
    for (char c : word.toCharArray()) {
        int idx = c - 'a';
        if (curr.children[idx] == null) {
            curr.children[idx] = new TrieNode();
        }
        curr = curr.children[idx];
    }
    curr.isEnd = true;
}
// Time: O(L) | Space: O(L) worst case (no shared prefix)
```

### Insert Trace

```
Insert "app":
  root → a (create) → p (create) → p (create, mark isEnd=true)

Insert "apple":
  root → a (exists) → p (exists) → p (exists) → l (create) → e (create, mark isEnd=true)

Insert "apt":
  root → a (exists) → p (exists) → t (create, mark isEnd=true)
```

## Search

```java
public boolean search(String word) {
    TrieNode curr = root;
    for (char c : word.toCharArray()) {
        int idx = c - 'a';
        if (curr.children[idx] == null) return false;
        curr = curr.children[idx];
    }
    return curr.isEnd;  // must be marked as complete word
}
// Time: O(L) | Space: O(1)
```

## StartsWith (Prefix Search)

```java
public boolean startsWith(String prefix) {
    TrieNode curr = root;
    for (char c : prefix.toCharArray()) {
        int idx = c - 'a';
        if (curr.children[idx] == null) return false;
        curr = curr.children[idx];
    }
    return true;  // any node at end of prefix is fine
}
// Time: O(L) | Space: O(1)
```

### Difference: search vs startsWith

```
Trie contains: {"apple", "app"}

search("app")       → true  (isEnd=true for "app")
search("ap")        → false (isEnd=false)
startsWith("ap")    → true  (prefix exists)
startsWith("apple") → true
startsWith("apz")   → false
```

## Delete

```java
public boolean delete(String word) {
    return delete(root, word, 0);
}

private boolean delete(TrieNode node, String word, int depth) {
    if (node == null) return false;

    if (depth == word.length()) {
        if (!node.isEnd) return false;  // word not found
        node.isEnd = false;
        return isLeaf(node);  // true → parent can delete this node
    }

    int idx = word.charAt(depth) - 'a';
    boolean shouldDelete = delete(node.children[idx], word, depth + 1);

    if (shouldDelete) {
        node.children[idx] = null;
        return !node.isEnd && isLeaf(node);  // delete this node too if safe
    }
    return false;
}

private boolean isLeaf(TrieNode node) {
    for (TrieNode child : node.children)
        if (child != null) return false;
    return true;
}
// Time: O(L) | Space: O(L) recursion
```

## Complete Trie Implementation

```java
public class Trie {
    private static class TrieNode {
        TrieNode[] children = new TrieNode[26];
        boolean isEnd;
    }

    private final TrieNode root = new TrieNode();

    public void insert(String word) {
        TrieNode curr = root;
        for (char c : word.toCharArray()) {
            int i = c - 'a';
            if (curr.children[i] == null) curr.children[i] = new TrieNode();
            curr = curr.children[i];
        }
        curr.isEnd = true;
    }

    public boolean search(String word) {
        TrieNode node = findNode(word);
        return node != null && node.isEnd;
    }

    public boolean startsWith(String prefix) {
        return findNode(prefix) != null;
    }

    private TrieNode findNode(String str) {
        TrieNode curr = root;
        for (char c : str.toCharArray()) {
            int i = c - 'a';
            if (curr.children[i] == null) return null;
            curr = curr.children[i];
        }
        return curr;
    }

    // Get all words with given prefix
    public List<String> autocomplete(String prefix) {
        List<String> result = new ArrayList<>();
        TrieNode node = findNode(prefix);
        if (node != null) dfs(node, new StringBuilder(prefix), result);
        return result;
    }

    private void dfs(TrieNode node, StringBuilder path, List<String> result) {
        if (node.isEnd) result.add(path.toString());
        for (int i = 0; i < 26; i++) {
            if (node.children[i] != null) {
                path.append((char)('a' + i));
                dfs(node.children[i], path, result);
                path.deleteCharAt(path.length() - 1);
            }
        }
    }

    // Count words with given prefix
    public int countWithPrefix(String prefix) {
        return autocomplete(prefix).size();
    }
}
```

## Advanced Trie Patterns

### Pattern 1: Word Search II (Trie + DFS on Grid)

```java
// LC 212 — find all words from dictionary in a 2D board
public List<String> findWords(char[][] board, String[] words) {
    Trie trie = new Trie();
    for (String w : words) trie.insert(w);

    Set<String> result = new HashSet<>();
    int m = board.length, n = board[0].length;

    for (int i = 0; i < m; i++)
        for (int j = 0; j < n; j++)
            dfs(board, i, j, trie.root, new StringBuilder(), result);

    return new ArrayList<>(result);
}

private void dfs(char[][] board, int r, int c, Trie.TrieNode node,
                 StringBuilder path, Set<String> result) {
    if (r < 0 || r >= board.length || c < 0 || c >= board[0].length) return;
    char ch = board[r][c];
    if (ch == '#') return;  // visited
    int idx = ch - 'a';
    if (node.children[idx] == null) return;

    node = node.children[idx];
    path.append(ch);
    if (node.isEnd) result.add(path.toString());

    board[r][c] = '#';
    dfs(board, r+1, c, node, path, result);
    dfs(board, r-1, c, node, path, result);
    dfs(board, r, c+1, node, path, result);
    dfs(board, r, c-1, node, path, result);
    board[r][c] = ch;
    path.deleteCharAt(path.length() - 1);
}
```

### Pattern 2: Design Search Autocomplete System

```java
// LC 642 — return top 3 most frequent historical searches
class AutocompleteSystem {
    private final Map<String, Integer> freq = new HashMap<>();
    private final Trie trie = new Trie();
    private StringBuilder current = new StringBuilder();

    public AutocompleteSystem(String[] sentences, int[] times) {
        for (int i = 0; i < sentences.length; i++) {
            freq.put(sentences[i], times[i]);
            trie.insert(sentences[i]);
        }
    }

    public List<String> input(char c) {
        if (c == '#') {
            String sentence = current.toString();
            freq.merge(sentence, 1, Integer::sum);
            trie.insert(sentence);
            current = new StringBuilder();
            return new ArrayList<>();
        }
        current.append(c);
        String prefix = current.toString();
        List<String> candidates = trie.autocomplete(prefix);
        candidates.sort((a, b) -> {
            int diff = freq.getOrDefault(b, 0) - freq.getOrDefault(a, 0);
            return diff != 0 ? diff : a.compareTo(b);
        });
        return candidates.subList(0, Math.min(3, candidates.size()));
    }
}
```

### Pattern 3: XOR Trie (Maximum XOR)

```java
// LC 421 — Maximum XOR of Two Numbers
// Store binary representation of numbers in a trie
class XORTrie {
    private static class Node {
        Node[] ch = new Node[2];
    }
    private final Node root = new Node();

    public void insert(int num) {
        Node curr = root;
        for (int i = 31; i >= 0; i--) {
            int bit = (num >> i) & 1;
            if (curr.ch[bit] == null) curr.ch[bit] = new Node();
            curr = curr.ch[bit];
        }
    }

    public int maxXOR(int num) {
        Node curr = root;
        int xor = 0;
        for (int i = 31; i >= 0; i--) {
            int bit = (num >> i) & 1;
            int want = 1 - bit;  // we want the opposite bit to maximize XOR
            if (curr.ch[want] != null) {
                xor |= (1 << i);
                curr = curr.ch[want];
            } else {
                curr = curr.ch[bit];
            }
        }
        return xor;
    }
}

public int findMaximumXOR(int[] nums) {
    XORTrie trie = new XORTrie();
    for (int n : nums) trie.insert(n);
    int max = 0;
    for (int n : nums) max = Math.max(max, trie.maxXOR(n));
    return max;
}
// Time: O(32n) = O(n) | Space: O(32n) = O(n)
```

### Problem 2: Add and Search Word ⭐⭐⭐⭐

**LeetCode 211** | Medium — supports '.' as wildcard

```java
public boolean searchWithWildcard(String word) {
    return searchHelper(root, word, 0);
}

private boolean searchHelper(TrieNode node, String word, int idx) {
    if (idx == word.length()) return node.isEnd;
    char c = word.charAt(idx);

    if (c == '.') {
        for (TrieNode child : node.children) {
            if (child != null && searchHelper(child, word, idx + 1)) return true;
        }
        return false;
    } else {
        int i = c - 'a';
        return node.children[i] != null && searchHelper(node.children[i], word, idx + 1);
    }
}
// Time: O(26^L) worst case for all wildcards | Space: O(L)
```

### Problem 5: Replace Words ⭐⭐⭐

**LeetCode 648** | Medium

```java
public String replaceWords(List<String> dictionary, String sentence) {
    Trie trie = new Trie();
    for (String root : dictionary) trie.insert(root);

    StringBuilder sb = new StringBuilder();
    for (String word : sentence.split(" ")) {
        if (sb.length() > 0) sb.append(' ');
        sb.append(shortestPrefix(trie, word));
    }
    return sb.toString();
}

private String shortestPrefix(Trie trie, String word) {
    TrieNode curr = trie.root;
    StringBuilder prefix = new StringBuilder();
    for (char c : word.toCharArray()) {
        int i = c - 'a';
        if (curr.children[i] == null) break;
        curr = curr.children[i];
        prefix.append(c);
        if (curr.isEnd) return prefix.toString();  // found root
    }
    return word;
}
// Time: O(sum of word lengths) | Space: O(sum of dictionary lengths)
```

## Key Takeaways

- Trie is optimal for prefix-related problems — O(L) per operation regardless of dictionary size.
- Use array `children[26]` for lowercase letters (fast), `HashMap` for general character sets (flexible).
- `search` checks `isEnd`; `startsWith` just checks if path exists — this is the key difference.
- For delete, use post-order recursion and clean up leaf nodes bottom-up.
- XOR Trie: insert 32-bit numbers bit by bit to find max XOR in O(32n).
- Trie + DFS is the canonical approach for Word Search II — prune the DFS using trie structure.
