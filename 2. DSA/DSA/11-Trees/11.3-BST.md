# 11.3 Binary Search Tree (BST)

A BST maintains the invariant that all left descendants are smaller and all right descendants are larger — this ordering enables O(log n) search on balanced trees and unlocks elegant solutions for order-statistic problems.

## BST Property

```
For every node N:
  All values in left subtree  < N.val
  All values in right subtree > N.val

        8
       / \
      3   10
     / \    \
    1   6    14
       / \   /
      4   7 13

Check: 3 < 8 ✓, 10 > 8 ✓, 6 ∈ (3,8) ✓, 13 ∈ (10,14) ✓
```

### Inorder gives sorted order

```
Inorder of BST above: 1 3 4 6 7 8 10 13 14
```

### Complexity

| Operation | Balanced BST | Worst (Skewed) |
|-----------|-------------|----------------|
| Search | O(log n) | O(n) |
| Insert | O(log n) | O(n) |
| Delete | O(log n) | O(n) |
| Min/Max | O(log n) | O(n) |
| Successor | O(log n) | O(n) |

## Search

```java
// Recursive
public TreeNode search(TreeNode root, int val) {
    if (root == null || root.val == val) return root;
    return val < root.val
        ? search(root.left, val)
        : search(root.right, val);
}

// Iterative (preferred — no recursion overhead)
public TreeNode searchIterative(TreeNode root, int val) {
    while (root != null && root.val != val) {
        root = val < root.val ? root.left : root.right;
    }
    return root;
}
// Time: O(h) | Space: O(1) iterative
```

## Insert

```java
// Recursive — returns modified root
public TreeNode insert(TreeNode root, int val) {
    if (root == null) return new TreeNode(val);
    if (val < root.val) root.left  = insert(root.left,  val);
    else if (val > root.val) root.right = insert(root.right, val);
    // if val == root.val: ignore duplicate (or handle as needed)
    return root;
}

// Iterative
public TreeNode insertIterative(TreeNode root, int val) {
    TreeNode newNode = new TreeNode(val);
    if (root == null) return newNode;

    TreeNode parent = null, curr = root;
    while (curr != null) {
        parent = curr;
        curr = val < curr.val ? curr.left : curr.right;
    }

    if (val < parent.val) parent.left  = newNode;
    else                  parent.right = newNode;
    return root;
}
// Time: O(h) | Space: O(h) recursive / O(1) iterative
```

## Delete

Three cases:

```
Case 1: Node is a leaf → just remove it.
Case 2: Node has one child → replace node with that child.
Case 3: Node has two children → replace node with inorder successor
        (smallest in right subtree), then delete successor from right subtree.

        Delete 3 from:         Result:
             8                      8
            / \                    / \
           3   10      →          4   10
          / \    \                 \    \
         1   6    14               6    14
            / \   /               / \   /
           4   7 13              5   7 13
```

```java
public TreeNode delete(TreeNode root, int val) {
    if (root == null) return null;

    if (val < root.val) {
        root.left = delete(root.left, val);
    } else if (val > root.val) {
        root.right = delete(root.right, val);
    } else {
        // Node to delete found
        if (root.left == null)  return root.right;  // Case 1 & 2
        if (root.right == null) return root.left;   // Case 2

        // Case 3: two children — find inorder successor
        TreeNode successor = findMin(root.right);
        root.val = successor.val;                          // copy value
        root.right = delete(root.right, successor.val);   // delete successor
    }
    return root;
}

private TreeNode findMin(TreeNode node) {
    while (node.left != null) node = node.left;
    return node;
}
// Time: O(h) | Space: O(h)
```

## Floor and Ceiling

```
Floor(k)   = largest value in BST that is ≤ k
Ceiling(k) = smallest value in BST that is ≥ k

BST: {1, 3, 4, 6, 7, 8, 10}
floor(5)   = 4
ceiling(5) = 6
floor(6)   = 6
ceiling(6) = 6
```

```java
// Floor: largest value ≤ key
public TreeNode floor(TreeNode root, int key) {
    if (root == null) return null;
    if (root.val == key) return root;

    if (root.val > key) return floor(root.left, key);

    // root.val < key: floor might be in right subtree
    TreeNode right = floor(root.right, key);
    return right != null ? right : root;
}

// Ceiling: smallest value ≥ key
public TreeNode ceiling(TreeNode root, int key) {
    if (root == null) return null;
    if (root.val == key) return root;

    if (root.val < key) return ceiling(root.right, key);

    // root.val > key: ceiling might be in left subtree
    TreeNode left = ceiling(root.left, key);
    return left != null ? left : root;
}
// Time: O(h) | Space: O(h)
```

## Kth Smallest Element

### Approach 1: Inorder Traversal (collect all)

```java
public int kthSmallest(TreeNode root, int k) {
    List<Integer> inorder = new ArrayList<>();
    inorderCollect(root, inorder);
    return inorder.get(k - 1);
}

private void inorderCollect(TreeNode node, List<Integer> list) {
    if (node == null) return;
    inorderCollect(node.left, list);
    list.add(node.val);
    inorderCollect(node.right, list);
}
// Time: O(n) | Space: O(n)
```

### Approach 2: Early-Stop Inorder (optimal)

```java
// LC 230 — stop after visiting k nodes
private int count = 0, result = 0;

public int kthSmallest(TreeNode root, int k) {
    count = k;
    inorderStop(root);
    return result;
}

private void inorderStop(TreeNode node) {
    if (node == null) return;
    inorderStop(node.left);
    if (--count == 0) { result = node.val; return; }
    inorderStop(node.right);
}
// Time: O(h + k) | Space: O(h)
```

### Approach 3: Iterative Inorder

```java
public int kthSmallestIterative(TreeNode root, int k) {
    Deque<TreeNode> stack = new ArrayDeque<>();
    TreeNode curr = root;
    while (curr != null || !stack.isEmpty()) {
        while (curr != null) { stack.push(curr); curr = curr.left; }
        curr = stack.pop();
        if (--k == 0) return curr.val;
        curr = curr.right;
    }
    return -1;
}
// Time: O(h + k) | Space: O(h)
```

## Validate BST

### Approach 1: Range Checking (Optimal)

```java
// LC 98 — each node must be in valid (min, max) range
public boolean isValidBST(TreeNode root) {
    return validate(root, Long.MIN_VALUE, Long.MAX_VALUE);
}

private boolean validate(TreeNode node, long min, long max) {
    if (node == null) return true;
    if (node.val <= min || node.val >= max) return false;
    return validate(node.left,  min, node.val)
        && validate(node.right, node.val, max);
}
// Time: O(n) | Space: O(h)
// Use Long to handle Integer.MIN_VALUE / MAX_VALUE edge cases
```

### Approach 2: Inorder Check

```java
// Inorder must be strictly increasing
private long prev = Long.MIN_VALUE;

public boolean isValidBSTInorder(TreeNode root) {
    if (root == null) return true;
    if (!isValidBSTInorder(root.left)) return false;
    if (root.val <= prev) return false;  // not strictly increasing
    prev = root.val;
    return isValidBSTInorder(root.right);
}
```

## BST to Sorted Array & Back

### BST → Sorted Array (Inorder)

```java
public int[] bstToArray(TreeNode root) {
    List<Integer> list = new ArrayList<>();
    inorder(root, list);
    return list.stream().mapToInt(Integer::intValue).toArray();
}

private void inorder(TreeNode node, List<Integer> list) {
    if (node == null) return;
    inorder(node.left, list);
    list.add(node.val);
    inorder(node.right, list);
}
```

### Sorted Array → BST (Balanced)

```java
// LC 108 — Convert Sorted Array to BST
public TreeNode sortedArrayToBST(int[] nums) {
    return build(nums, 0, nums.length - 1);
}

private TreeNode build(int[] nums, int lo, int hi) {
    if (lo > hi) return null;
    int mid = lo + (hi - lo) / 2;
    TreeNode root = new TreeNode(nums[mid]);
    root.left  = build(nums, lo, mid - 1);
    root.right = build(nums, mid + 1, hi);
    return root;
}
// Time: O(n) | Space: O(log n) recursion
```

### BST Iterator (Lazy Inorder)

```java
// LC 173 — next() and hasNext() in O(h) time, O(h) space
class BSTIterator {
    private Deque<TreeNode> stack = new ArrayDeque<>();

    public BSTIterator(TreeNode root) {
        pushLeft(root);
    }

    public int next() {
        TreeNode node = stack.pop();
        pushLeft(node.right);
        return node.val;
    }

    public boolean hasNext() {
        return !stack.isEmpty();
    }

    private void pushLeft(TreeNode node) {
        while (node != null) {
            stack.push(node);
            node = node.left;
        }
    }
}
```

## Common Problems

### Problem 1: Lowest Common Ancestor of BST ⭐⭐⭐⭐⭐

**LeetCode 235** | Easy

```java
// Use BST property — no need to visit all nodes
public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
    while (root != null) {
        if (p.val < root.val && q.val < root.val)
            root = root.left;
        else if (p.val > root.val && q.val > root.val)
            root = root.right;
        else
            return root;  // split point = LCA
    }
    return null;
}
// Time: O(h) | Space: O(1)
```

```java
// Reverse inorder (right → root → left), accumulate running sum
private int runningSum = 0;

public TreeNode bstToGst(TreeNode root) {
    if (root == null) return null;
    bstToGst(root.right);
    runningSum += root.val;
    root.val = runningSum;
    bstToGst(root.left);
    return root;
}
// Time: O(n) | Space: O(h)
```

### Problem 6: Two Sum in BST ⭐⭐⭐

**LeetCode 653** | Easy

```java
// Inorder collect + two pointer
public boolean findTarget(TreeNode root, int k) {
    List<Integer> sorted = new ArrayList<>();
    inorder(root, sorted);
    int lo = 0, hi = sorted.size() - 1;
    while (lo < hi) {
        int sum = sorted.get(lo) + sorted.get(hi);
        if (sum == k) return true;
        else if (sum < k) lo++;
        else hi--;
    }
    return false;
}
```

## Key Takeaways

- BST property: `left < node < right` — use this to avoid visiting all nodes.
- Delete with two children: replace with inorder successor (min of right subtree).
- Validate BST with range `(min, max)` — simpler and handles edge cases better than inorder prev check.
- `kthSmallest`: iterative inorder with early stop is both O(h+k) time and O(h) space.
- LCA in BST is O(h) with no recursion — exploit the ordering instead of brute-force tree traversal.
- Sorted array → BST: always pick `mid = lo + (hi - lo) / 2` to build a balanced tree.
