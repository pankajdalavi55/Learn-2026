## Polynomial Hashing

### Concept

Map a string to a number using a polynomial over a prime base.

```
hash(s) = s[0]*p^(n-1) + s[1]*p^(n-2) + ... + s[n-1]*p^0  (mod MOD)

Common values:
  p   = 31 (for lowercase letters) or 131 (for all ASCII)
  MOD = 1_000_000_007 (large prime) or 998_244_353
```

### Implementation

```java
public class PolynomialHash {
    private static final long P = 31;
    private static final long MOD = 1_000_000_007L;

    public static long hash(String s) {
        long h = 0, power = 1;
        for (int i = s.length() - 1; i >= 0; i--) {
            h = (h + (s.charAt(i) - 'a' + 1) * power) % MOD;
            power = (power * P) % MOD;
        }
        return h;
    }

    public static void main(String[] args) {
        System.out.println(hash("abc"));   // 2976
        System.out.println(hash("abc"));   // same → equal strings → same hash
        System.out.println(hash("bac"));   // different
    }
}
```

### Prefix Hash Array

```java
public class PrefixHash {
    private final long[] h;     // prefix hashes
    private final long[] pow;   // powers of P
    private static final long P = 31, MOD = 1_000_000_007L;

    public PrefixHash(String s) {
        int n = s.length();
        h = new long[n + 1];
        pow = new long[n + 1];
        pow[0] = 1;
        for (int i = 0; i < n; i++) {
            h[i + 1] = (h[i] * P + (s.charAt(i) - 'a' + 1)) % MOD;
            pow[i + 1] = (pow[i] * P) % MOD;
        }
    }

    // Hash of substring s[l..r] (0-indexed, inclusive) — O(1)
    public long query(int l, int r) {
        return (h[r + 1] - h[l] * pow[r - l + 1] % MOD + MOD * MOD) % MOD;
    }
}
```

**Usage:**
```java
PrefixHash ph = new PrefixHash("abcabc");
System.out.println(ph.query(0, 2) == ph.query(3, 5));  // true, both "abc"
```

## Rolling Hash

### Concept

Slide a window of fixed size across the string, updating the hash in O(1) by removing the leftmost character and adding the rightmost character.

```
Window "abc" over "xabcd":
  hash("abc") = a*p² + b*p + c

Slide right to "bcd":
  hash("bcd") = hash("abc") * p - a*p³ + d
              = (old_hash - a * p^(len-1)) * p + d  (mod MOD)
```

### Implementation

```java
public class RollingHash {
    private static final long BASE = 31;
    private static final long MOD  = 1_000_000_007L;

    private long windowHash;
    private long highPow;  // BASE^(len-1) % MOD
    private final int len;

    public RollingHash(String s, int windowLen) {
        this.len = windowLen;
        highPow = 1;
        for (int i = 0; i < windowLen - 1; i++) {
            highPow = (highPow * BASE) % MOD;
        }

        // Hash of first window
        windowHash = 0;
        for (int i = 0; i < windowLen; i++) {
            windowHash = (windowHash * BASE + (s.charAt(i) - 'a' + 1)) % MOD;
        }
    }

    /**
     * Roll window: remove leftmost char, add rightmost char.
     * outChar = character leaving window
     * inChar  = character entering window
     */
    public long roll(char outChar, char inChar) {
        windowHash = (windowHash - (outChar - 'a' + 1) * highPow % MOD + MOD) % MOD;
        windowHash = (windowHash * BASE + (inChar - 'a' + 1)) % MOD;
        return windowHash;
    }

    public long current() { return windowHash; }
}
```

**Rabin-Karp pattern matching:**
```java
public List<Integer> search(String text, String pattern) {
    int n = text.length(), m = pattern.length();
    List<Integer> result = new ArrayList<>();
    if (m > n) return result;

    RollingHash rh = new RollingHash(text, m);
    long patHash = new RollingHash(pattern, m).current();

    if (rh.current() == patHash && text.substring(0, m).equals(pattern)) {
        result.add(0);
    }

    for (int i = 1; i <= n - m; i++) {
        rh.roll(text.charAt(i - 1), text.charAt(i + m - 1));
        if (rh.current() == patHash &&
            text.substring(i, i + m).equals(pattern)) {
            result.add(i);  // verify to handle hash collision
        }
    }
    return result;
}
```

### Complexity

| Step | Time | Space |
|------|------|-------|
| Preprocessing | O(m) | O(1) |
| Each slide | O(1) | O(1) |
| Full text scan | O(n) | O(1) |
| Total (Rabin-Karp) | O(n + m) avg | O(1) |

## Double Hashing

### Concept

Use **two independent hash functions** with different bases and moduli. Two strings match only if **both** hashes match, reducing false-positive probability to ≈ 1 / (MOD1 × MOD2).

```
Single hash false-positive rate: ~1 / 10^9
Double hash false-positive rate: ~1 / 10^18 (negligible)
```

### Implementation

```java
public class DoubleHash {
    private static final long P1 = 31,  MOD1 = 1_000_000_007L;
    private static final long P2 = 37,  MOD2 = 998_244_353L;

    private final long[] h1, h2, pow1, pow2;

    public DoubleHash(String s) {
        int n = s.length();
        h1 = new long[n + 1];   h2 = new long[n + 1];
        pow1 = new long[n + 1]; pow2 = new long[n + 1];
        pow1[0] = pow2[0] = 1;

        for (int i = 0; i < n; i++) {
            int v = s.charAt(i) - 'a' + 1;
            h1[i+1] = (h1[i] * P1 + v) % MOD1;
            h2[i+1] = (h2[i] * P2 + v) % MOD2;
            pow1[i+1] = (pow1[i] * P1) % MOD1;
            pow2[i+1] = (pow2[i] * P2) % MOD2;
        }
    }

    // Returns (hash1, hash2) pair for s[l..r]
    public long[] query(int l, int r) {
        long q1 = (h1[r+1] - h1[l] * pow1[r-l+1] % MOD1 + MOD1 * 2) % MOD1;
        long q2 = (h2[r+1] - h2[l] * pow2[r-l+1] % MOD2 + MOD2 * 2) % MOD2;
        return new long[]{q1, q2};
    }

    public boolean equal(int l1, int r1, int l2, int r2) {
        long[] a = query(l1, r1), b = query(l2, r2);
        return a[0] == b[0] && a[1] == b[1];
    }
}
```

## String Hashing Applications

### 1. Longest Repeated Substring (Binary Search + Hash)

```java
public String longestRepeatedSubstring(String s) {
    int n = s.length();
    int lo = 0, hi = n - 1, result = 0;
    String bestMatch = "";

    while (lo <= hi) {
        int mid = (lo + hi) / 2;
        String match = check(s, mid);
        if (match != null) {
            result = mid;
            bestMatch = match;
            lo = mid + 1;
        } else {
            hi = mid - 1;
        }
    }
    return bestMatch;
}

private String check(String s, int len) {
    if (len == 0) return "";
    Set<Long> seen = new HashSet<>();
    long base = 31, mod = 1_000_000_007L;
    long h = 0, pow = 1;

    for (int i = 0; i < len; i++) {
        h = (h * base + s.charAt(i) - 'a' + 1) % mod;
        if (i > 0) pow = (pow * base) % mod;
    }
    seen.add(h);

    for (int i = 1; i + len <= s.length(); i++) {
        h = (h - (s.charAt(i-1) - 'a' + 1) * pow % mod + mod) % mod;
        h = (h * base + s.charAt(i + len - 1) - 'a' + 1) % mod;
        if (!seen.add(h)) return s.substring(i, i + len);
    }
    return null;
}
```

### 2. Count Distinct Substrings

```java
public int countDistinctSubstrings(String s) {
    int n = s.length();
    Set<Long> distinct = new HashSet<>();
    long base = 31, mod = 1_000_000_007L;

    for (int len = 1; len <= n; len++) {
        long h = 0, pow = 1;
        for (int i = 0; i < len; i++) {
            h = (h * base + s.charAt(i) - 'a' + 1) % mod;
            if (i > 0) pow = pow * base % mod;
        }
        distinct.add(h * 1000 + len);  // encode (hash, len) to reduce collisions

        for (int i = 1; i + len <= n; i++) {
            h = (h - (s.charAt(i-1) - 'a' + 1) * pow % mod + mod) % mod;
            h = (h * base + s.charAt(i + len - 1) - 'a' + 1) % mod;
            distinct.add(h * 1000 + len);
        }
    }
    return distinct.size();
}
```

## Common Problems

### Problem 1: Longest Duplicate Substring ⭐⭐⭐⭐

**LeetCode 1044** | Hard

```java
public String longestDupSubstring(String s) {
    int n = s.length();
    int lo = 1, hi = n - 1;
    String result = "";

    while (lo <= hi) {
        int mid = (lo + hi) / 2;
        String dup = getdup(s, mid);
        if (dup != null) {
            result = dup;
            lo = mid + 1;
        } else {
            hi = mid - 1;
        }
    }
    return result;
}

private String getdup(String s, int len) {
    long base = 26, mod = (1L << 32);
    long h = 0, pow = 1;
    Map<Long, List<Integer>> seen = new HashMap<>();

    for (int i = 0; i < len; i++) {
        h = (h * base + s.charAt(i)) % mod;
        if (i > 0) pow = (pow * base) % mod;
    }
    seen.computeIfAbsent(h, k -> new ArrayList<>()).add(0);

    for (int i = 1; i + len <= s.length(); i++) {
        h = (h - s.charAt(i-1) * pow % mod + mod) % mod;
        h = (h * base + s.charAt(i + len - 1)) % mod;
        List<Integer> positions = seen.get(h);
        if (positions != null) {
            String candidate = s.substring(i, i + len);
            for (int pos : positions) {
                if (s.substring(pos, pos + len).equals(candidate)) return candidate;
            }
        }
        seen.computeIfAbsent(h, k -> new ArrayList<>()).add(i);
    }
    return null;
}
// Time: O(n log n) | Space: O(n)
```

### Problem 3: Implement strStr (Rabin-Karp) ⭐⭐⭐

**LeetCode 28** | Easy

```java
public int strStr(String haystack, String needle) {
    int n = haystack.length(), m = needle.length();
    if (m > n) return -1;
    long base = 31, mod = 1_000_000_007L, pow = 1;

    long nh = 0, hh = 0;
    for (int i = 0; i < m; i++) {
        nh = (nh * base + needle.charAt(i)) % mod;
        hh = (hh * base + haystack.charAt(i)) % mod;
        if (i > 0) pow = (pow * base) % mod;
    }

    if (nh == hh && haystack.substring(0, m).equals(needle)) return 0;

    for (int i = 1; i + m <= n; i++) {
        hh = (hh - haystack.charAt(i-1) * pow % mod + mod) % mod;
        hh = (hh * base + haystack.charAt(i + m - 1)) % mod;
        if (hh == nh && haystack.substring(i, i + m).equals(needle)) return i;
    }
    return -1;
}
// Time: O(n + m) avg | Space: O(1)
```

## Key Takeaways

- Polynomial hash maps strings to numbers; prefix hash arrays enable O(1) substring comparison.
- Rolling hash slides a window in O(1): subtract outgoing char's contribution, multiply by base, add incoming char.
- Always use `verify with equals()` after a hash match to handle collisions (false positives).
- Double hashing (two hash functions) reduces false-positive probability to practically zero.
- Binary search on length + rolling hash = O(n log n) solution for "longest X substring" problems.
- Choose MOD as a large prime (1e9+7) and BASE as a prime larger than the alphabet size.
