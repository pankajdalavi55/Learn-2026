## Introduction

### What is a HashSet?

A **HashSet** is a collection that stores **unique elements** with O(1) average add/remove/contains operations. Internally it is backed by a `HashMap` where each element is a key and the value is a dummy constant.

```
HashSet<Integer>: {3, 1, 4, 1, 5, 9, 2, 6}
After insertion:  {1, 2, 3, 4, 5, 6, 9}   ← duplicates removed
```

### When to Use

| Use Case | Data Structure |
|----------|---------------|
| Fast duplicate detection | HashSet |
| Visited tracking (BFS/DFS) | HashSet |
| Intersection / union of sets | HashSet |
| Sorted unique elements | TreeSet |
| Insertion-ordered unique elements | LinkedHashSet |

---

## How HashSet Works Internally

```
HashSet<K> is backed by HashMap<K, PRESENT>
where PRESENT = new Object() (shared dummy value)

add(k)       → map.put(k, PRESENT)
contains(k)  → map.containsKey(k)
remove(k)    → map.remove(k)

Same bucket/collision rules as HashMap apply.
```

## Core Operations

```java
import java.util.*;

public class HashSetBasics {
    public static void main(String[] args) {
        Set<Integer> set = new HashSet<>();

        // add — O(1), returns false if already present
        set.add(10);
        set.add(20);
        set.add(30);
        boolean added = set.add(10);  // false, duplicate
        System.out.println(set);      // [20, 10, 30] (order not guaranteed)

        // contains — O(1)
        System.out.println(set.contains(20));  // true
        System.out.println(set.contains(99));  // false

        // remove — O(1)
        set.remove(20);

        // size / isEmpty
        System.out.println(set.size());     // 2
        System.out.println(set.isEmpty());  // false

        // iterate
        for (int x : set) {
            System.out.print(x + " ");
        }

        // addAll, retainAll, removeAll
        Set<Integer> other = new HashSet<>(Arrays.asList(10, 40, 50));
        set.addAll(other);    // union
        set.retainAll(other); // intersection
        set.removeAll(other); // difference
    }
}
```

## Set Mathematics

```java
Set<Integer> A = new HashSet<>(Arrays.asList(1, 2, 3, 4, 5));
Set<Integer> B = new HashSet<>(Arrays.asList(3, 4, 5, 6, 7));

// Union  A ∪ B
Set<Integer> union = new HashSet<>(A);
union.addAll(B);
// {1, 2, 3, 4, 5, 6, 7}

// Intersection  A ∩ B
Set<Integer> intersection = new HashSet<>(A);
intersection.retainAll(B);
// {3, 4, 5}

// Difference  A − B
Set<Integer> difference = new HashSet<>(A);
difference.removeAll(B);
// {1, 2}

// Symmetric Difference  A △ B
Set<Integer> symDiff = new HashSet<>(union);
symDiff.removeAll(intersection);
// {1, 2, 6, 7}

// Subset check  A ⊆ B
boolean isSubset = B.containsAll(A);  // false
```

### Complexity of Set Operations

| Operation | HashSet | TreeSet |
|-----------|---------|---------|
| add | O(1) avg | O(log n) |
| contains | O(1) avg | O(log n) |
| remove | O(1) avg | O(log n) |
| union (addAll) | O(n) | O(n log n) |
| intersection (retainAll) | O(n) | O(n log n) |

## Common HashSet Patterns

### Pattern 1: Duplicate Detection

```java
// Check if array has duplicates (LC 217)
public boolean containsDuplicate(int[] nums) {
    Set<Integer> seen = new HashSet<>();
    for (int n : nums) {
        if (!seen.add(n)) return true;  // add returns false if already present
    }
    return false;
}
// Time: O(n) | Space: O(n)
```

### Pattern 2: Visited Set in BFS/DFS

```java
// Word Ladder pattern
public int ladderLength(String begin, String end, List<String> wordList) {
    Set<String> wordSet = new HashSet<>(wordList);
    Set<String> visited = new HashSet<>();
    Queue<String> queue = new LinkedList<>();

    queue.offer(begin);
    visited.add(begin);
    int steps = 1;

    while (!queue.isEmpty()) {
        int size = queue.size();
        for (int i = 0; i < size; i++) {
            String word = queue.poll();
            if (word.equals(end)) return steps;
            char[] arr = word.toCharArray();
            for (int j = 0; j < arr.length; j++) {
                char original = arr[j];
                for (char c = 'a'; c <= 'z'; c++) {
                    arr[j] = c;
                    String next = new String(arr);
                    if (wordSet.contains(next) && !visited.contains(next)) {
                        visited.add(next);
                        queue.offer(next);
                    }
                }
                arr[j] = original;
            }
        }
        steps++;
    }
    return 0;
}
```

### Pattern 3: Longest Consecutive Sequence

```java
// LC 128 — key insight: only start from sequence beginnings
public int longestConsecutive(int[] nums) {
    Set<Integer> numSet = new HashSet<>();
    for (int n : nums) numSet.add(n);

    int maxLen = 0;
    for (int n : numSet) {
        if (!numSet.contains(n - 1)) {   // n is sequence start
            int current = n, len = 1;
            while (numSet.contains(current + 1)) {
                current++;
                len++;
            }
            maxLen = Math.max(maxLen, len);
        }
    }
    return maxLen;
}
// Time: O(n) | Space: O(n)
```

### Pattern 4: Happy Number

```java
// LC 202 — detect cycle with visited set
public boolean isHappy(int n) {
    Set<Integer> seen = new HashSet<>();
    while (n != 1) {
        if (!seen.add(n)) return false;  // cycle detected
        int sum = 0;
        while (n > 0) {
            int d = n % 10;
            sum += d * d;
            n /= 10;
        }
        n = sum;
    }
    return true;
}
```

### Pattern 5: Remove Duplicates While Preserving Order

```java
// Use LinkedHashSet to deduplicate and keep insertion order
public int[] removeDuplicates(int[] arr) {
    Set<Integer> seen = new LinkedHashSet<>();
    for (int x : arr) seen.add(x);
    return seen.stream().mapToInt(Integer::intValue).toArray();
}
```

## HashSet vs TreeSet vs LinkedHashSet

| Feature | HashSet | TreeSet | LinkedHashSet |
|---------|---------|---------|---------------|
| Order | None | Sorted (natural/comparator) | Insertion order |
| add/contains/remove | O(1) avg | O(log n) | O(1) avg |
| Null support | 1 null allowed | No null | 1 null allowed |
| Backed by | HashMap | Red-Black Tree | LinkedHashMap |
| Use when | Fast O(1) ops | Need sorted | Need order + fast ops |

```java
// TreeSet — sorted unique elements
TreeSet<Integer> treeSet = new TreeSet<>();
treeSet.add(5); treeSet.add(1); treeSet.add(3);
System.out.println(treeSet);             // [1, 3, 5]
System.out.println(treeSet.floor(4));    // 3 (largest ≤ 4)
System.out.println(treeSet.ceiling(2));  // 3 (smallest ≥ 2)
System.out.println(treeSet.first());     // 1
System.out.println(treeSet.last());      // 5

// LinkedHashSet — insertion-ordered unique elements
LinkedHashSet<String> lhs = new LinkedHashSet<>();
lhs.add("banana"); lhs.add("apple"); lhs.add("cherry");
System.out.println(lhs);  // [banana, apple, cherry]
```

## Common Problems

### Problem 1: Contains Duplicate ⭐⭐⭐⭐⭐

**LeetCode 217** | Easy

```java
public boolean containsDuplicate(int[] nums) {
    Set<Integer> seen = new HashSet<>();
    for (int n : nums) if (!seen.add(n)) return true;
    return false;
}
// Time: O(n) | Space: O(n)
```

### Problem 2: Intersection of Two Arrays ⭐⭐⭐

**LeetCode 349** | Easy

```java
public int[] intersection(int[] nums1, int[] nums2) {
    Set<Integer> set1 = new HashSet<>();
    for (int n : nums1) set1.add(n);

    Set<Integer> result = new HashSet<>();
    for (int n : nums2) if (set1.contains(n)) result.add(n);

    return result.stream().mapToInt(Integer::intValue).toArray();
}
// Time: O(m + n) | Space: O(min(m, n))
```

### Problem 3: Longest Consecutive Sequence ⭐⭐⭐⭐⭐

**LeetCode 128** | Medium

```java
public int longestConsecutive(int[] nums) {
    Set<Integer> numSet = new HashSet<>();
    for (int n : nums) numSet.add(n);
    int maxLen = 0;
    for (int n : numSet) {
        if (!numSet.contains(n - 1)) {
            int len = 1;
            while (numSet.contains(n + len)) len++;
            maxLen = Math.max(maxLen, len);
        }
    }
    return maxLen;
}
// Time: O(n) | Space: O(n)
```

### Problem 4: Single Number ⭐⭐⭐

**LeetCode 136** | Easy (Set approach, XOR is O(1) space but set is instructive)

```java
public int singleNumber(int[] nums) {
    Set<Integer> seen = new HashSet<>();
    for (int n : nums) {
        if (!seen.add(n)) seen.remove(n);  // remove on second occurrence
    }
    return seen.iterator().next();
}
// Time: O(n) | Space: O(n)
```

## Key Takeaways

- `HashSet.add(x)` returns `false` if already present — use this for duplicate detection in one line.
- `contains` is O(1); never use `indexOf` on a list when a set will do.
- Use `LinkedHashSet` when insertion order must be preserved.
- Use `TreeSet` when you need sorted order or floor/ceiling/first/last navigation.
- Set operations (`addAll`, `retainAll`, `removeAll`) implement union, intersection, and difference.
- In BFS/DFS, always use a `visited` HashSet to avoid re-visiting nodes.
